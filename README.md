# HSC Focus Room

A dedicated study application for HSC Science students in Bangladesh with exactly 7 months left for exam preparation.

## Features

- **Distraction-Free Study Environment**: Fullscreen focus mode with Pomodoro timer
- **Study Tracker System**: Daily checklists, progress tracking, and statistics
- **7-Month Smart Planner**: Automatic study plan distribution across subjects
- **Subject-Wise Study Space**: Separate sections for each subject and paper
- **Authentication**: Secure login with email/Google and data persistence
- **Responsive Design**: Optimized for mobile devices

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (Auth + PostgreSQL)
- **Charts**: Recharts
- **Deployment**: Vite build ready for Vercel
